# 📊 Analytics Engineering & Strategic Intelligence Portfolio

Bienvenido a mi repositorio central de proyectos. Aquí encontrarás la intersección entre la **ingeniería de datos avanzada** y la **inteligencia estratégica** aplicada a la toma de decisiones.

## 🚀 Sobre este Repositorio
Este espacio sirve como prueba técnica de mis capacidades para transformar datos crudos en activos estratégicos. Mi enfoque se centra en la eliminación de silos de información y la optimización de procesos mediante el uso de **Microsoft Fabric**, **SQL** y **Power BI**.

### 📂 Contenido Destacado

#### 1. [Comando de Inteligencia Estratégica.pdf](./Strategic_Intelligence_Command.pdf)
* **Descripción:** Presentación ejecutiva sobre metodologías de inteligencia competitiva y análisis predictivo.
* **Caso de Estudio:** Auditoría de rendimiento comercial con identificación de brechas críticas.

#### 2. [Scripts de SQL Operativo]
* **Habilidades:** Modelado de esquemas relacionales, lógica de negocio compleja y optimización de consultas.
* **Impacto:** Automatización de procesos de cálculo para rendimiento por departamento.

* **Análisis y Modelado:** SQL, Python, DAX.
